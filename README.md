#### For inital use
- Insatll Git Version Control for CLI
- Sign in using
```bash
    git config --global user.name "USERNMAE"
``` 
```bash
    git config --global user.email "EMAIL"
````
- Clone a repository, first copy the _url_ then 

```bash
    git clone   URL
```
- Make your changes
- To add all files 
 ```bash
    git add .
 ```
 - To give commit message
 ```bash
    git commit -m "Your message"
 ```
 - To push the changes
 ```bash
    git push
 ```
